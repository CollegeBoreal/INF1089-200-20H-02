## :one: Créer votre projet dans un répertoire :id:

* Example

```
$ mkdir 300098957 && cd 300098957
```

## :two: Créer un fichier README.md expliquant votre projet

```
$ touch README.md
```

```python
if __name__== "__main__":
   main()
```
